# Domain Glossary

| المصطلح | المعنى العملي داخل النظام |
|---|---|
| System | تعريف النظام الفني الذي يحكم قواعد الإنتاج لعائلة قطاعات معينة. |
| ProfileFamily | فئة عامة للقطاع (Sliding/Fixed/Casement...). |
| ProfileVariant | نسخة محددة قابلة للاستخدام من القطاع داخل المشروع. |
| ProfileGeometry | التوصيف الهندسي للقطاع (أبعاد/مناطق/سطوح تركيب). |
| SemanticZone | منطقة دلالية داخل القطاع مثل frame/sash/bead/glass. |
| InterfaceDefinition | تعريف نقاط التركيب بين القطاعات والاكسسوارات والزجاج. |
| Accessory | عنصر إكسسوار مرتبط بمخزون وقاعدة حساب. |
| CompatibilityRule | قاعدة تحقق توافق بين الكيانات قبل التنفيذ. |
| AssemblyRule | قاعدة تنظيم تركيب العناصر (ثابت/متحرك/تقسيمات). |
| CutRule | قاعدة القص (زاوية القص، سماكة السكينة، الهالك الفني، طول الخام). |
| GlazingRule | قاعدة حساب ألواح الزجاج وأبعادها وكمياتها. |
| MachiningTemplate | قالب عمليات تصنيع معيارية (ثقوب/فتحات/تجهيزات). |
| BOMItem | سطر مادة موحد في المخرجات (قطاع/زجاج/إكسسوار/تشغيل). |
| BOMRule | آلية تحويل عنصر مُكوَّن إلى أسطر BOM. |
| ConfiguredElement | عنصر منتج مُكوَّن فعليًا بمواصفات نهائية. |
| AssemblyTree | تمثيل هرمي موحد للتجميعة بدل النماذج المتفرقة. |
| Opening | فتحة منتجة (نافذة/باب/جزء واجهة) بإحداثيات وأبعاد ونوع. |
| Sash | جزء متحرك داخل العنصر (درفة). |
| Fixed Panel | جزء ثابت غير متحرك داخل العنصر. |
| Mullion | قاطع داخلي يقسم الفتحة إلى أجزاء. |
| Bead (Brkluz) | قطاع تثبيت الزجاج. |
| Physical Role | دور تصنيعي للسطر مثل `frame_w` أو `glass_sash`. |
| Material Flow | حالة استهلاك/عجز المواد مقارنة بالمخزون. |
| Shortage | كمية/مساحة ناقصة يجب توفيرها بالشراء. |
| Cutting Map | ترتيب القص على الأعواد/الألواح لتقليل الهدر. |
| Workshop Operation | بند تشغيل داخلي (تجميع/تركيب/زجاج/تشطيب). |
| Measurement-only | وضع مقايسة بلا عقد: لا استهلاك مخزون فعلي. |
| Contract-linked | وضع تنفيذ بعقد: استهلاك مخزون + عجز + شراء + متابعة مالية. |
